# Can I contribute?

You are basically always free to contribute and I am always happy for contributions. If you want to do something drastic, first contact me on discord (Username mayaqq) or ping me in my discord https://discord.gg/hue. Though please read the specifications below before doing anything.
# Contributing with translations

I am happy for any new translations or corrections in them! If you want to contribute to them, most of the lang files are stored here.

If you want to specifically contribute to the english translation, you will have to change entries in the datagen file here.


# Porting this mod to different versions/mod loaders
While I do not disallow this, I would much prefer if you contacted us (as stated above) and discussed this topic with me first. I would also prefer if the mod itself stayed on this repository and on the official modpages on curseforge and modrinth.
